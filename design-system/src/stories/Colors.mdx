## Usage

In your components or page layouts, use the standard page background color for all main content areas.

```tsx
<Box sx={{ bgcolor: 'background.default' }}>
  {/* Content here */}
</Box>
```

The `PageLayout` component now defaults to `#FAFCFC`.

# Layout Standards

## Header Border

To maintain a clean and integrated look, headers should **not** have a bottom border. This is handled by default in the `PageLayout` component using the `hideHeaderBorder` prop set to `true`.

```tsx
<TopNav
  sx={{ borderBottom: 'none' }}
  // ... other props
/>
```
